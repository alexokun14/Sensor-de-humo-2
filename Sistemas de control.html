Linea de profundización: Sistemas de control

Este proyecto se encuentra en la línea de profundización sistema de control ya que, según X.Álvarez (2004), define el sistema de control como un ente que recibe unas acciones externas o variables de entrada, y cuya respuesta a estas acciones externas son las denominadas variables de salida. Se encuentra dentro de esta línea de profundización porque el sensor de humo se encarga de recibir acciones externas y el módulo wifi esp8266 controlado por el microcontrolador Arduino 1 es la denominada variable de salida, dando así un ingreso a la línea de profundización. El resultado es una conexión a través de un sensor de alarma para que tenga lugar en un control adecuado, comenzando con el estado de activación y resonancia, proporcionando mayor efectividad para el funcionamiento.

El sistema de control pretende conseguir, mediante el manejo de las variables de control, un poder sobre las variables de salida para que estas alcancen unos valores prefijados. Se caracteriza por la presencia de elementos que permiten influir en el funcionamiento y garantizar la estabilidad del sistema para evitar daños por incendio. Los sistemas de control digital presentan ventajas como una menor susceptibilidad al deterioro por el tiempo o factores del entorno, y componentes menos sensibles a los ruidos y vibraciones en las señales, proporcionando mayor seguridad a las personas ante un posible incendio.

Los elementos principales de un sistema de control que permiten su manejo son:

  • Sensor: permite conocer los valores de las variables medidas del sistema
  • Controlador: emplea los valores determinados por los sensores
  • Actuador: ejecuta la acción calculada por el controlador y modifica las variables de control
